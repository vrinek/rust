An atomically reference counted wrapper for shared state.

Example

In this example, a large vector of floats is shared between several tasks. With simple pipes, without Arc, a copy would have to be made for each task.

use std::sync::Arc; use std::thread::Thread; fn main() { let numbers: Vec<_> = (0..100u32).map(|i| i as f32).collect(); let shared_numbers = Arc::new(numbers); for _ in 0..10 { let child_numbers = shared_numbers.clone(); Thread::spawn(move || { let local_numbers = child_numbers.as_slice(); // Work with the local numbers }); } }
use std::sync::Arc;
use std::thread::Thread;

fn main() {
    let numbers: Vec<_> = (0..100u32).map(|i| i as f32).collect();
    let shared_numbers = Arc::new(numbers);

    for _ in 0..10 {
        let child_numbers = shared_numbers.clone();

        Thread::spawn(move || {
            let local_numbers = child_numbers.as_slice();

            // Work with the local numbers
        });
    }
}

fn drop(&mut self)

Drops the Arc<T>.

This will decrement the strong reference count. If the strong reference count becomes zero and the only other references are Weak<T> ones, drops the inner value.

Examples
